# 小众点评 - 前端

毕业设计

---

## 项目设置

### 下载依赖

```sh
npm install
```

### 运行开发环境

```sh
npm run dev
```

### 构建

```sh
npm run build
```

### 使用 [ESLint](https://eslint.org/) 格式化代码

```sh
npm run lint
```

## 环境变量说明

| 变量名             | 描述        | 默认值 |
|-----------------|-----------|-----|
| `VITE_BASE_URL` | 前端页面地址。   | -   |
| `VITE_BASE_API` | 后端接口地址前缀。 | -   |
| `VITE_CDN_URL`  | CDN 地址前缀  | -   |
